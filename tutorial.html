tmc.py usage

Installation

If you have root access

You will need python3.2+ and pip3 installed. 

sudo pip3 install tmc

If you don't have root access

You will need python3.2+ and python-virtualenv installed. 

curl -L https://rawgit.com/JuhaniImberg/tmc.py/master/tmcpy-inst.sh | bash

Getting started

After you have installed tmc.py, you should run 

tmc configure

It will prompt you for the TMC servers address (for example https://tmc.mooc.fi/mooc/, username, password, a course and a download path for the exercises.

Downloading

If you have already completed some exercises you can download the remaining ones with 

tmc download

You can download all of the currently available exercises with 

tmc download --all

The exercises will be placed in the courses download path.

Selecting a exercise

In tmc.py you always have a selected exercise. The selection can be changed with 

tmc select

You can move to the next exercise with 

tmc next

You can move to the previous exercise with 

tmc previous

You can see what exercise you have selected with 

tmc current

Or you can see all of the exercises in the current course with 

tmc ls

Testing

You can test the currently selected exercise with 

tmc test

If testing fails tmc test will print the errors. However if it's successful it will just print OK.

Submitting

You can submit the selected exercise with 

tmc submit

You can also send the submission to TMC pastebin with 

tmc submit --paste

You can also request a code review with 

tmc submit --review
